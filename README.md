Sistem Administrasi Distributor Pupuk berbasis Next.js dengan Firebase Firestore.

## Fitur
- Login dengan ID Kantor dan Password
- Input Proforma Invoice dengan upload PDF
- Rekap Proforma Invoice (view, edit, delete, export Excel)
- Input Stock Gudang dengan perhitungan otomatis
- Laporan Stock Gudang (view, edit, delete, export Excel)
- Responsive design untuk semua ukuran layar

## Teknologi
- Next.js 14
- TypeScript
- Tailwind CSS
- Firebase (Auth, Firestore, Storage)
- XLSX untuk export Excel
